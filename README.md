# Co-Pack

A contract packaging idle/simulation game where your workforce actually matters.

## Packages

- `packages/engine` -- Core simulation engine (pure TypeScript, no UI)
- `packages/web` -- Employbridge web demo (React + Vite)
- `packages/mobile` -- Consumer game (React Native + Expo) *(Phase 3)*

## Getting Started

```bash
npm install
npm run dev:web
```
